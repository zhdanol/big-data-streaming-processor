# Big Data Domain Extractor (Streaming Methodology)

## Описание
Высокопроизводительный скрипт на Python для извлечения доменных имен из сверхбольших текстовых файлов (50 ГБ+). 
Проект разработан с фокусом на информационную безопасность и эффективное управление ресурсами системы.

## Основная проблема
Стандартные методы обработки текста часто требуют загрузки файла в оперативную память. При работе с файлами объемом 50 ГБ+ это приводит к переполнению RAM и аварийному завершению процесса.

## Решение и Технологии
- **Метод:** Потоковая обработка данных (Streaming). Чтение файла производится построчно/блоками, что гарантирует стабильное потребление RAM (не более 50-100 МБ) вне зависимости от размера входного файла.
- **Оптимизация:** Использование предварительно скомпилированных регулярных выражений (модуль re) для ускорения поиска на миллионах строк.
- **Отказоустойчивость:** Реализована обработка ошибок кодировки (errors='ignore'), что критично при анализе дампов данных или логов.

## Как использовать
1. Поместите файл для анализа в корневую папку проекта.
2. В скрипте укажите путь к входному файлу.
3. Запустите через терминал:
   python main.py
